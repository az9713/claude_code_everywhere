## Part 1: What is Termius?

### The Simple Explanation

**Termius** is an app that lets you control your computer's command line from your phone.

Think of it as a **remote control for your computer's terminal**, but instead of buttons for volume and channels, you have a keyboard for typing commands.

### How to Use Ctrl Combinations

To press `Ctrl + C` (cancel a running command):

1. **Tap and HOLD** the `Ctrl` button (it highlights)
2. While holding, **tap `C`** on the keyboard
3. Release

For `Ctrl + B` (tmux prefix):
1. Tap and hold `Ctrl`
2. Tap `B`
3. Release
4. Then tap the next key (like `D` for detach)

### Swipe for More Keys

In Termius, you can **swipe left** on the special keys bar to reveal more:

## Part 8: Getting to WSL and tmux

When you connect to a Windows PC, you land in Windows Command Prompt or PowerShell. To get to your tmux session, you need a few more steps.

### Step 8.1: Enter WSL (Linux)

Type:
```
wsl
```

Press Enter (tap the Return key on keyboard).

Your prompt changes from:
```
C:\Users\simon>
```

To:
```
simon@COMPUTER:~$
```

You're now in Linux!

### Step 8.2: Attach to Your tmux Session

Type:
```
tmux attach -t claude -d
```

Press Enter.

Now you're in your Claude Code session (or whatever was running in tmux)!

## Part 10: Disconnecting Properly

### Method 1: Exit Everything Gracefully

If you want to completely disconnect:

1. **If in Claude Code**: Exit Claude Code (Ctrl+C or type "exit")

2. **If in tmux**: Detach with `Ctrl+B`, then `d`
   - This keeps tmux running!

3. **Exit WSL**: Type `exit`

4. **Close Termius connection**: Tap the X or swipe away

### Method 2: Just Detach (Recommended)

If you want to come back later:

1. **In tmux**: Press `Ctrl+B`, then `d`

2. You're back at the WSL prompt

3. You can close Termius - everything stays running!

## Part 11: Troubleshooting

### Problem: "Connection Refused"

```
Error: Connection refused
```

**Causes:**
- SSH server not running on the computer
- Wrong IP address
- Computer is off or asleep

**Solutions:**
1. Make sure the computer is on
2. Verify SSH is running (on Windows: check Services)
3. Verify the IP address is correct
4. Make sure Tailscale is connected (if using Tailscale)

### Problem: "Authentication Failed"

```
Error: Authentication failed
```

**Causes:**
- Wrong username
- Wrong password
- Typo in either

**Solutions:**
1. Double-check username (case-sensitive on Linux!)
2. Re-enter password carefully
3. Verify credentials work by logging into the computer directly

### Problem: "Host Key Verification Failed"

```
Error: Host key verification failed
```

**Cause:**
The computer's SSH key has changed since you last connected.

**Solution:**
1. Edit the host in Termius
2. Clear the cached host key
3. Connect again and accept the new key

### Problem: Connection Drops Frequently

**Causes:**
- Weak internet connection
- Phone going to sleep
- Network switching (WiFi to mobile)

**Solutions:**
1. Use tmux! It survives connection drops.
2. Check your internet connection
3. Adjust phone sleep settings

### Problem: Can't Type Special Characters

**Cause:**
Special keyboard not visible

**Solution:**
Look for the special keys bar above the keyboard. Swipe left for more keys.
